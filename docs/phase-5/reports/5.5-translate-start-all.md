## MANUAL SMOKE (for human)

### Basic path
- Navigate to Step 5 (Translate) on a draft with 2+ target languages
- All languages show "Not started" / per-language Start buttons as before
- "Start All" button visible at top-right above the language list
- Click Start All → button shows spinner + becomes disabled
- First language flips to "Translating…" within ~1s
- Second language flips to "Translating…" ~1s after first
- Both proceed to completion via existing SignalR/WorkflowSubscriber
- After all done, "Start All" button is disabled (hasStartable = false)

### Edge case — one language already Validated
- Open draft where 1 of 2 languages is Validated
- Click Start All
- Only the not-yet-translated language starts; Validated stays Validated

### Edge case — single language draft
- Loop iterates once; button works identically to per-language Start
