## Post-Ship Operational Step (§5.7 task 7)

Before triggering the first Demo deploy, set these Railway env vars on the Demo API service:

- `Seed__SuperUser__Email`
- `Seed__SuperUser__Password`
- `Seed__Admin__Email`
- `Seed__Admin__Password`

ASP.NET Core maps `__` separators to `:` in configuration — these map to `Seed:SuperUser:Email` etc. Missing values log a warning and skip that account; they do not throw or block startup.
