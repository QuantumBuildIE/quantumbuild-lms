# QuantumBuild LMS

Learning Management System for workplace safety training and compliance.

## Tech Stack
- **Backend:** .NET 9, ASP.NET Core, Entity Framework Core, PostgreSQL
- **Frontend:** Next.js 16, React 19, TypeScript, shadcn/ui, Tailwind CSS
- **Background Jobs:** Hangfire
- **File Storage:** Cloudflare R2
- **AI Translation:** Claude API

## Features
- Video-based training with anti-skip tracking
- AI-generated content sections and quizzes
- Multi-language translations (sections, quizzes, subtitles, slideshows)
- Course management with sequential completion
- Certificate generation (PDF)
- Refresher scheduling with email reminders
- Employee self-service portal
- Admin compliance dashboard
