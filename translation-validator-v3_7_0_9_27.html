ARCHITECTURE NOTE
This Tool = authoring / validation only (runs in browser) · CertifiedIQ Railway = delivery platform (serves pre-validated translations to workers) · Cloudflare Worker = proxy holding DeepL / Gemini / DeepSeek keys securely
Translations are validated once here, then stored on Railway. Workers receive pre-approved content — no live AI calls on-site. Quality investment happens at authoring time, not delivery time.
